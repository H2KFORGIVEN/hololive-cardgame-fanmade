維護 web/data/tournaments.json — 大賽基本資料（event / date / location）+ 官方使用率統計。 這裡填的資料會顯示在前台「大賽」頁的 event 標題 + 使用率長條圖。

❓ 這頁能填什麼？

大賽基本資料（每一列）

  • event：大賽名稱，要跟 deck_codes.json 裡的 event 完全一致（前台用這個對上 decks）
  • date：大賽日期 YYYY-MM-DD
  • location：會場名稱

使用率（optional，官方公布後再填）

  • 點每列的「使用率」按鈕展開 modal
  • scope 要填四個語系（zh-TW / en / ja / fr），例：「預選ラウンド」
  • source：資料來源（例：hololive OFFICIAL CARD GAME）
  • rates：推し使用率列表，加總 ~= 100%，最後通常一列「その他」補尾數
event date location 使用率統計

存檔後叫 Claude 幫你 push-deploy，或自己跑 ./scripts/push-deploy.sh。

如果這場還沒官方統計怎辦？

不用填這個 modal，直接關掉就好。沒填使用率的 event 在前台顯示時只會有標題 + 牌組，不會顯示長條圖。

有填過但想整個刪掉 → 點下方「清除使用率」按鈕。
