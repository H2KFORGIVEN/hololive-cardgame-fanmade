每一列是一個大賽牌組。存檔後下次跑 scraper 會自動從 Bushiroad API 抓牌譜詳細內容。

❓ code 欄怎麼填？

code 不是自己編的。它是 Bushiroad Deck Log 網站給每個「有人上傳過」的牌組一個 5 碼唯一 ID， URL 長這樣：https://decklog-en.bushiroad.com/ja/view/44ZSC ← 最後那段就是 code。

要從哪裡拿到 code？

  • 官方 X 帳號 @hololive_OCG po 大賽結果時常附 Deck Log 連結
  • 選手自己的 X / Discord 貼文
  • 攻略社群（ホロカ攻略ギルド、Vanholo 等）整理的大賽結果

還沒有 code 的大賽怎麼辦？

勾選該列的「placeholder」— 就算沒 code 也能先記下 event / 日期 / 名次 / oshi。 之後選手公開牌組再回來填 code、取消 placeholder。

code title oshi source event event_date placement ? 牌譜
